# Diabetes Prediction using Machine Learning

This project predicts whether a patient has diabetes using machine learning.

## Dataset
The dataset contains medical information such as glucose level, blood pressure, BMI, insulin level, and age.

## Algorithm Used
Support Vector Machine

## Libraries Used
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Project Steps
1. Load dataset
2. Data preprocessing
3. Train-test split
4. Train machine learning model
5. Evaluate model accuracy
