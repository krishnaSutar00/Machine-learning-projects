# Sonar Rock vs Mine Prediction

This project uses a Machine Learning model to classify sonar signals as either Rock or Mine.

## Dataset
Sonar Dataset from UCI Machine Learning Repository.

## Algorithm Used
Logistic Regression

## Libraries Used
- NumPy
- Pandas
- Scikit-learn

## Steps in the Project
1. Load dataset
2. Data preprocessing
3. Train-test split
4. Train Logistic Regression model
5. Evaluate model accuracy

## Result
The model predicts whether a sonar signal corresponds to a rock or a mine.
